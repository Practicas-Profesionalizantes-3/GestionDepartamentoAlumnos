# 🎓 Sistema de Gestión para Departamento de Alumnos

## 📌 Descripción

Aplicación web orientada a optimizar la comunicación entre los alumnos y el departamento de alumnos. Permite gestionar trámites, recibir notificaciones en tiempo real y acceder a información institucional desde un entorno centralizado y accesible.

## 🚀 Objetivo

Digitalizar y agilizar los procesos académicos, reduciendo tiempos de espera, eliminando burocracia y evitando desplazamientos innecesarios.

## 🧩 Funcionalidades Principales

* 📄 **Trámites y solicitudes online** con seguimiento de estado
* 🔔 **Notificaciones en tiempo real** (académicas, personales y generales)
* 📢 **Cartelera virtual** con avisos, eventos y novedades
* 📂 **Gestor de documentos** con descarga y búsqueda
* ✍️ **Firma digital** y adjuntos en solicitudes
* 🔍 **Búsqueda avanzada** y filtros de información

## 👤 Gestión de Usuarios

* Registro, login y recuperación de contraseña
* Edición de perfil
* Acceso seguro a funcionalidades

## ⚙️ Requerimientos No Funcionales

* ⏱️ Tiempo de respuesta ≤ 3 segundos
* 📶 Disponibilidad ≥ 99.9%
* 🔐 Seguridad y estándares (ISO/IEC 27001)
* 💡 Interfaz intuitiva y fácil de usar
* 🔗 Integración con servicios externos (ej: calendarios)

## 🎯 Finalidad

Mejorar la eficiencia administrativa, reducir el uso de papel y brindar una experiencia digital moderna para estudiantes.
